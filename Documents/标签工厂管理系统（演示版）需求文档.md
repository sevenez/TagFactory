## 1. 系统概述

### 1.1 项目背景

标签工厂管理系统（演示版）是一个专注于用户标签管理、群体分析和数据治理的综合性平台演示版本。系统旨在向用户展示完整的标签管理功能，帮助企业了解如何建立完善的用户画像体系。

### 1.2 系统目标



* 展示统一的标签管理体系，支持多维度用户画像构建

* 提供灵活的群体创建和分析功能演示

* 支持多数据源接入和数据质量监控展示

* 提供标准化的 API 接口演示

* 确保演示数据的安全性和完整性

### 1.3 系统架构

系统采用 B/S 架构，基于 Web 浏览器访问，支持主流浏览器兼容。演示版无需用户登录，直接展示功能界面。

### 6.1 数据安全

#### 6.1.1 数据加密



* 传输加密：所有 HTTP 请求使用 HTTPS 协议

* 存储加密：敏感数据使用 AES-256 加密算法

#### 6.1.2 数据脱敏



* 手机号：显示前 3 位和后 4 位，中间用 \* 代替

* 身份证号：显示前 6 位和后 4 位，中间用 \* 代替

* 邮箱地址：用户名部分显示前 2 位，其余用 \* 代替

### 6.2 演示数据保护
